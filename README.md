# SpawnStar
A C++ SDL3 app that procedurally generates stars on your screen.

# How to run!

1. Get a C++ compiler, You can just get VS Code and use the C/C++ extenstion.

2. Get the newest version of CMake: [Download CMake][eagain]

3. Install this repo

4. Make a directory called "build" inside the root folder.

5. Go into build

6. Run `cmake ..` inside using a shell

7. Now run `cmake --build .` inside using a shell

8. Find SpawnStar.exe in build

9. Make sure you have SDL3.dll in the same directory as SpawnStar.exe

- SpawnStar.exe can be in any directory now, as long as SDL3.dll is in it.
