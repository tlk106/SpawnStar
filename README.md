# How to build.

1. Get a C++ compiler

2. Get the newest version of CMake: [Download CMake][cmake_slop]

3. Install this repo

4. Make a directory called "build" inside the root folder.

5. Go into build

6. Run `cmake ..` inside using a shell

7. Now run `cmake --build .` inside using a shell

8. Find the SpawnStar executable in build

- SpawnStar.exe can be in any directory now, as long as SDL3.dll is in it.
