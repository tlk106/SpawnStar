# SpawnStar
A C++ SDL3 app that procedurally generates stars on your screen.

# How to run!

1. Get the newest version of CMake: [Download CMake][e]

2. Install this repo

3. Make a directory called "build".

4. Go into build

5. Run `cmake ..` inside using a shell

6. Now run `cmake --build .` inside using a shell

7. Find main.exe

8. Make sure you have SDL3.dll in the same directory as main.exe
